## This project is an open source C++ framework designed to:

---

1. Automate robot arms projects with ease, specifically targeting non-developers.
2. (Demo preparation tool) Help plan the construction of a project by first visualizing the expected outcome.

It leverages a C++ backend for performance and efficiency, and a React frontend for a dynamic and interactive user interface
with a crow server to handle HTTP requests and implements CORS to facilitate secure communication
between the frontend and backend.

## Prerequisites

---

Before executing the project, ensure you have installed the required dependencies:

- Install 'Microsoft Runtime Packages'

  -> Link : https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170

- Unzip 'Prerequisites.zip' & execute 'C++ SDK_Framework.exe'

## Quick Start Guide

---

### Installation

Follow these steps to install the necessary components:

<Stepper>
   <StepperItem title="Install Node.js">
      Ensure you have Node.js installed.
   </StepperItem>

<StepperItem title="Clone The Repository">
  Clone our GitHub repository to your local machine.

```bash
git clone https://github.com/CutshionFramework/Documentation_robot_contoller.git
```

</StepperItem>

<StepperItem title="Install Dependencies">
  Navigate to the project directory and install dependencies.

```bash
cd Documentation_robot_controller
npm install
#or
npm i
```

</StepperItem>

<StepperItem title="Run The Project">
```bash   
npm start   
```

</StepperItem>

</Stepper>
